使用說明

  1. Clone 此專案 
    git clone https://github.com/sinchichou/ZUOSUN-Crypto.git
cd ZUOSUN-Crypto
  2. 安裝必要套件 
    pip install -r requirements.txt
  3. 配置環境變數
    • 交易所 API 金鑰
    • 資料庫連線資訊
  4. 執行主程式 
    python main.py
